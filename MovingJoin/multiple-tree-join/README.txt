Usage: 
(1) Open the project with Visual Studio (version >= 2005).
(2) Compile and run the code on an exemplary dataset.
(3) Modify the experimental settings in main.cpp to observe the code in various settings. There is also a switch (one statement #define REPORT_RESULT) in main.cpp to control the reporting of the join result.
